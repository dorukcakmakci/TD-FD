README
------
TD-FD is a context-free language that operates on predicate calculus domain.
Further details about design decisions and a BNF description of TD-FD can be found in "td-fd.report.pdf".
yacc configuration file of TD-FD: "td-fd.yacc.y"
lex configuration file of TD-FD: "td-fd.lex.l"
a test code written in TD-FD: "td-fd.test.txt"

NOTE: TD-FD is developed as a course project for CS315(Programming Languages) course of Bilkent University.
